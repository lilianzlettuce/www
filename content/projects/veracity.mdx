# Veracity

A web application interface designed for charity searching and discovery, built to help users find and evaluate charitable organizations.

## Overview

Veracity was created to simplify the process of finding and researching charitable organizations. The application provided an intuitive interface for users to search, compare, and learn about various charities and their impact.

## Key Features

- **Charity Search**: Comprehensive database of charitable organizations
- **Information Display**: Detailed profiles and impact metrics
- **User-Friendly Interface**: Intuitive design for easy navigation
- **API Integration**: Connected to charity databases for real-time data
- **Responsive Design**: Accessible across all devices

## Technical Implementation

Built with modern web technologies, the application integrated with external APIs to provide comprehensive charity information and search capabilities.

## Project Status

The application is no longer running due to API restrictions, but the project demonstrates proficiency in web development and API integration.

## Future Possibilities

- Integration with alternative charity databases
- Enhanced search and filtering capabilities
- User reviews and ratings system
- Donation tracking features
