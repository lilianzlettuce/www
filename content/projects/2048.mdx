# 2048

A browser-based implementation of the classic 2048 puzzle game, featuring original styling and multiple themes for an enhanced gaming experience.

## Overview

This project recreates the popular 2048 sliding tile puzzle game with custom styling and multiple visual themes. The implementation focuses on smooth gameplay and responsive design across all devices.

## Key Features

- **Classic Gameplay**: Faithful recreation of the original 2048 mechanics
- **Multiple Themes**: Various visual styles to customize the experience
- **Responsive Design**: Optimized for desktop and mobile play
- **Smooth Animations**: Fluid tile movements and transitions
- **Score Tracking**: Persistent high score system
- **Touch Controls**: Full support for mobile touch interaction

## Technical Implementation

Built with vanilla JavaScript, HTML, and CSS, the project demonstrates solid understanding of web development fundamentals and game logic implementation.

## Game Mechanics

The game follows the classic 2048 rules where players slide numbered tiles to combine them and reach the elusive 2048 tile, with strategic thinking and planning required for success.
