# ChipIn

A comprehensive household management application designed to simplify shared living through intelligent inventory tracking, expense management, and AI-powered meal planning.

## Overview

ChipIn addresses the common challenges of shared living by providing a centralized platform for managing household tasks, expenses, and meal planning. Built as a team project for Software Engineering I, it demonstrates collaborative development practices.

## Key Features

- **Shared Inventory Tracking**: Real-time updates for household items
- **Expense Splitting**: Automated calculation and tracking of shared costs
- **AI Meal Planning**: Intelligent suggestions based on available ingredients
- **Collaborative Tools**: Multiple users can contribute and view updates
- **Responsive Design**: Works seamlessly across devices

## Technical Implementation

Built with React and Firebase, the app leverages cloud services for real-time collaboration and data persistence across multiple users.

## Team Development

This project was developed by a team of 4 students, demonstrating effective collaboration, version control, and software engineering best practices.
